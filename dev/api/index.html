API Reference

Discretization

HelmholtzBenchmark.discretize_helmholtz — Function
discretize_helmholtz(n::Int, k::Real) -> (A::SparseMatrixCSC{Float64,Int}, b::Vector{Float64})

Discretize the Helmholtz equation $(-∇²-k²)u = f$ on $[0,1]$ using second-order finite differences. Uses $n$ interior points (resulting in an $n \times n$ matrix). $f$ is set to a delta function at $x = 1/2$. Returns the system matrix A and the right-hand side vector b.
